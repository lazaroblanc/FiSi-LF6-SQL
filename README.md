Dieses Repository beinhaltet Lösungen zu Übungsaufgaben aus dem Berufsschulunterricht.
Ziel der Übungen war es den Grundlegenden SQL Syntax zu erlernen.

Die Aufgaben basierten auf der "Northwind Traders" Beispieldatenbank von Microsoft.

Ich habe diese Datenbank hier nicht in das Repository hochgeladen, da ich keine eindeutigen
Informationen zu der Lizenz der Datenbank finden konnte. Die Beispieldatenbank die wir
im Unterricht genutzt haben, wurde von unserem Lehrer bereitgestellt.

Die originale Version scheint sich hier zu befinden: <a href="https://github.com/microsoft/sql-server-samples/tree/master/samples/databases/northwind-pubs"> microsoft/sql-server-samples</a>
Allerdings ist diese für einen Microsoft SQL Server gedacht.

Unsere Übungen wurden auf einer MySQL Datenbank durchgeführt.
